## 🧠 What is AllergenSafe?

> ⚡ A next-gen AI system that scans food images and detects hidden allergens using deep learning.

💡 Built using Vision Transformers (ViT) + clinical allergen datasets  
📊 Designed for real-world safety & healthcare use  

---

## 🎥 System Flow
📸 Image Upload
↓
🧠 Vision Transformer Model
↓
🍽 Food Classification
↓
⚠️ Allergen Detection Engine
↓
📊 Confidence + Dashboard Output
---

## ✨ Key Features

### 🔍 AI Intelligence
- 📸 Image-based food recognition
- 🧠 Vision Transformer (ViT)
- 📈 Confidence scoring

### ⚠️ Safety First
- 🥜 Detects peanuts, gluten, dairy, shellfish & more
- 🧬 Clinical dataset mapping
- 🚨 Cross-contamination alerts

### 🎨 Premium UI
- 📊 Dashboard analytics
- 🗂 Scan history tracking
- 🔎 Smart filtering & search

## 🚀 Run Locally

### ⚙️ Backend Setup

```bash
source venv/bin/activate
uvicorn app.api:app --host 0.0.0.0 --port 8000
🎨 Frontend Setup
cd frontend
npm install
npm start

👉 UI: http://localhost:3000

```
